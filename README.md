
# Try Speech Studio üöÄ

This repository is to store the data and the result from the use of some of the tools that are available in the Speech Studio.

Speech Studio is a set of UI-based tools that lets you explore, build, and integrate features from Azure AI Speech. Speech Studio lets you try several service features and sample their returned data in a quick, straightforward manner.
